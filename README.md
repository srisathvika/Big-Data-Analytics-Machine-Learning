# Big-Data-Analytics-Machine-Learning

## Overview

This repository contains my coursework and assignments completed as part of the **Big Data Analytics and Machine Learning** module. The assignments cover big data processing, data analysis, and the implementation of various machine learning algorithms using PySpark and Python.

The purpose of this repository is to demonstrate practical applications of data analytics and machine learning concepts through hands-on exercises and projects.

---

## Topics Covered

### Big Data Processing
- Introduction to PySpark
- Spark DataFrames
- Data loading and preprocessing
- Data transformation and aggregation
- Exploratory data analysis

### Supervised Machine Learning
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forests
- Ensemble Learning Techniques
- Model Evaluation

### Unsupervised Machine Learning
- K-Means Clustering
- Cluster Evaluation
- Feature Engineering

### Advanced Topics
- Natural Language Processing (NLP)
- Recommendation Systems
- Additional advanced machine learning techniques
